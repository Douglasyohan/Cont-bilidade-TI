T1 Estatística Descritiva
▾

O que cai em prova:

  • Construção de tabela de distribuição de frequências (f, fr, fac)
  • Classe, amplitude de classe, ponto médio de cada classe
  • Histograma: identificar barras, associar ao contexto
  • Polígono de frequência: ponto médio das classes
  • Ogiva (frequência acumulada relativa)
⚠ Atenção especial: frequência relativa acumulada (frac) — muito cobrada em questões de percentil.

Frequências a saber:

SímboloSignificado
fFrequência absoluta
frFrequência relativa (f/n)
faFrequência acumulada
fraFrequência relativa acumulada
nTotal de observações

Amplitude da classe:

Amplitude de Classe (h)
h = (Xmax − Xmin) ÷ k

k ≈ √n (Regra da raiz)
k ≈ 1 + 3,3 · log(n) (Sturges)
Frequência Relativa
fr = f / n
Frequência Acumulada
fa_i = fa_(i-1) + f_i
Ponto Médio da Classe
Pm = (Li + Ls) / 2     onde Li=limite inferior, Ls=limite superior
Regra de Sturges (nº de classes)
k = 1 + 3,3 · log(n)
⚠ Erro #1: Calcular amplitude total em vez de amplitude da classe.
⚠ Erro #2: Não fechar a frequência acumulada em n (100%).
⚠ Erro #3: Confundir histograma (barras adjacentes) com gráfico de barras separadas.
💡 Dica de prova: Sempre verifique se Σf = n e Σfr = 1 (ou 100%). Se não bater, tem erro.
T2 Medidas de Tendência Central
▾
💡 Foco principal em prova: cálculo de média, mediana e moda para dados agrupados em classes.
MedidaDados BrutosDados AgrupadosQuando usar
MédiaΣx/nΣ(Pm·f)/nDistribuição simétrica
MedianaValor central ordenadoFórmula interpolaçãoDados assimétricos
ModaValor mais frequenteFórmula de CzuberValores repetidos

Identificando a classe mediana: Encontre a primeira classe cuja frequência acumulada (fa) ≥ n/2. Essa é a classe mediana.

Identificando a classe modal: É a classe com maior frequência absoluta (f). A moda pertence a essa classe.

Média Aritmética — Dados Brutos
x̄ = (x₁ + x₂ + ... + xₙ) / n = Σxᵢ / n
Média Aritmética — Dados Agrupados
x̄ = Σ(Pmᵢ · fᵢ) / n
Pm = ponto médio da classe = (Li + Ls) / 2
Mediana — Dados Agrupados (Interpolação)
Md = L_md + [(n/2 − F_ant) / f_md] · h

L_md = limite inferior da classe mediana
F_ant = frequência acumulada ANTES da classe mediana
f_md = frequência da classe mediana
h = amplitude da classe
Moda — Czuber (dados agrupados)
Mo = L_mo + [Δ₁ / (Δ₁ + Δ₂)] · h

L_mo = limite inferior da classe modal
Δ₁ = f_mo − f_anterior
Δ₂ = f_mo − f_posterior
h = amplitude da classe
✅ Macete Czuber: Δ₁ e Δ₂ são sempre positivos. Se der negativo, você subtraiu errado.
⚠ Erro #1: Usar o limite superior em vez do inferior nas fórmulas de Md e Mo.
⚠ Erro #2: Usar a frequência relativa em vez da absoluta na fórmula da mediana.
⚠ Erro #3: Calcular Δ₁ como f_posterior − f_mo (sinal invertido).
⚠ Erro #4: Esquecer de usar o ponto médio na média de dados agrupados (usar limite inferior).
💡 Verificação: Em distribuição simétrica: Mo = Md = x̄. Em assimétrica positiva: Mo < Md < x̄.
T3 Medidas de Dispersão
▾
MedidaFórmula BrutaInterpreta como
Amplitude (A)Xmax − XminDispersão total
Variância (s²)Σ(xᵢ−x̄)²/(n−1)Dispersão quadrática média
Desvio Padrão (s)√s²Dispersão na mesma unidade dos dados
CV(s / x̄) · 100Dispersão relativa (%)
💡 CV como julgamento de homogeneidade: CV ≤ 15% = baixa dispersão (homogêneo) | CV 15–30% = média | CV > 30% = alta dispersão (heterogêneo)
Variância Amostral
s² = Σ(xᵢ − x̄)² / (n − 1)
Variância — Dados Agrupados
s² = Σ[fᵢ · (Pmᵢ − x̄)²] / (n − 1)
Desvio Padrão
s = √s²
Coeficiente de Variação
CV = (s / x̄) · 100%
Fórmula Computacional (variância — mais rápida)
s² = [Σxᵢ² − (Σxᵢ)²/n] / (n − 1)
⚠ Erro #1: Dividir por n em vez de (n−1) na variância amostral.
⚠ Erro #2: Não elevar ao quadrado os desvios — calcular Σ(xᵢ−x̄) em vez de Σ(xᵢ−x̄)².
⚠ Erro #3: Calcular CV sem multiplicar por 100 (esquecer o %).
⚠ Erro #4: Em dados agrupados, usar xᵢ em vez do ponto médio Pmᵢ.
T4 Probabilidade e Distribuição Normal
▾

Em provas de Contabilidade, o foco é na distribuição normal padronizada (Z) e não em probabilidade combinatória complexa.

ConceitoSignificado prático
Distribuição NormalCurva simétrica em forma de sino, média = mediana = moda
Padronização (Z)Transforma qualquer normal em N(0,1) para usar Tabela Z
Área sob a curvaRepresenta probabilidade. Área total = 1 (100%)
P(a < X < b)Diferença de áreas na tabela Z
💡 Regra empírica (68-95-99,7): μ±1σ → 68% | μ±2σ → 95% | μ±3σ → 99,7% dos dados.
Padronização Z
Z = (X − μ) / σ

X = valor observado
μ = média da população
σ = desvio padrão da população
Probabilidade entre dois valores
P(a < X < b) = Φ(Z_b) − Φ(Z_a)
onde Φ = área acumulada à esquerda na Tabela Z
Área à direita (cauda superior)
P(X > a) = 1 − Φ(Z_a)
Simetria da normal
P(Z < −z) = P(Z > z) = 1 − Φ(z)
⚠ Erro #1: Esquecer de subtrair 0,5 quando a tabela Z fornece área do centro ao Z (em vez de 0 ao Z).
⚠ Erro #2: Não verificar se a tabela é acumulada à esquerda ou do centro.
⚠ Erro #3: Inverter a subtração: P(a<X<b) = Φ(Za) − Φ(Zb) (ordem errada).
✅ Antes de usar a Tabela Z, sempre desenhe a curva normal e pinte a área que você quer. Isso evita 90% dos erros.

⚠ Erro #2: Não tirar raiz de n na fórmula do intervalo de confiança.
⚠ Erro #3: Usar Z = 1,96 para 90% de confiança (correto é 1,645).
⚠ Erro #4: Confundir "rejeita H₀" com "prova que H₁ é verdadeira". São coisas diferentes.

▾

💡 R² (coeficiente de determinação): R² = r² → percentual da variação de Y explicada por X. Ex: r=0,9 → R²=0,81 → 81% da variação explicada.

Regressão linear: Ŷ = a + bX  →  b é o coeficiente angular (quanto Y cresce para cada X adicional). a é o intercepto (valor de Y quando X=0).

Correlação de Pearson
r = [nΣXY − (ΣX)(ΣY)] / √{[nΣX² − (ΣX)²] · [nΣY² − (ΣY)²]}
Coeficiente Angular da Reta de Regressão
b = [nΣXY − (ΣX)(ΣY)] / [nΣX² − (ΣX)²]
Intercepto
a = Ȳ − b · X̄     onde Ȳ = ΣY/n e X̄ = ΣX/n
Equação da Reta de Regressão
Ŷ = a + b · X
Para prever: substitua X pelo valor desejado
Coeficiente de Determinação
R² = r² (variação de Y explicada pela reta)
⚠ Erro #1: Confundir r (correlação) com R² (determinação). São interpretações diferentes.
⚠ Erro #2: Usar a equação ŷ = a + bx com valores de X fora do intervalo amostral (extrapolação indevida).
⚠ Erro #3: Calcular b com o denominador errado (ΣX² em vez de nΣX²−(ΣX)²).
✅ Monte sempre uma tabela auxiliar com colunas: X, Y, X², Y², XY. Some cada coluna. Isso organiza todos os valores necessários.

QUESTÃO 03 · Distribuição Normal e Inferência (2,5 pts)

O custo de estoque diário de um atacadista segue distribuição normal com μ = R$ 25.000 e σ = R$ 3.500.
(a) P(custo > R$ 30.000);
(b) P(R$ 20.000 < custo < R$ 28.500);
(c) Um analista afirma que a média real é R$ 26.500. Com uma amostra de 49 dias, x̄ = R$ 25.800. Usando α = 5%, teste se há evidência de que a média é diferente de R$ 25.000. [σ = 3.500]

(a) Z = (30.000−25.000)/3.500 = 1,43 → P(Z>1,43) = 1−0,9236 = 0,0764 (7,64%)

(b) Za=(20.000−25.000)/3.500=−1,43 | Zb=(28.500−25.000)/3.500=1,00
P = Φ(1,00)−Φ(−1,43) = 0,8413−0,0764 = 0,7649 (76,49%)

(c) Teste de hipóteses:
H₀: μ = 25.000 | H₁: μ ≠ 25.000 (bicaudal)
Z_calc = (25.800−25.000)/(3.500/√49) = 800/500 = 1,60
Z_crítico para α=5% bicaudal = ±1,96
|1,60| < 1,96 → Não rejeita H₀. Não há evidência estatística suficiente de que a média difere de R$ 25.000.
