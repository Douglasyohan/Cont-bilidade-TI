# 📘 Static Study Platform — Accounting & Statistics

## 📌 Overview

This repository contains a static web platform built for reviewing Financial Accounting and supporting a structured study plan for Statistics.

The project is designed with a **static-first architecture**, prioritizing simplicity, performance, and zero backend dependency.

---

## 🎯 Purpose

* Provide a lightweight environment for educational content consumption
* Enable fast access to structured study materials
* Maintain a portable and easily deployable knowledge base

---

## 🏗️ Architecture

This project follows a **Static Site Architecture**:

* Client-side rendering only
* No server-side processing
* No database or persistent storage
* CDN-friendly deployment

---

## ⚙️ Tech Stack

* **HTML5** — Structure
* **CSS3** — Styling
* **JavaScript (optional)** — Interactivity
* **Hosting** — GitHub Pages / Netlify

---

## 📂 Project Structure

```
.
├── index.html              # Entry point
├── assets/
│   ├── css/               # Stylesheets
│   ├── js/                # Scripts
│   └── images/            # Static assets
├── accounting/            # Financial accounting content
├── statistics/            # Statistics study materials
└── exercises/             # Practice content
```

---

## 🚀 Deployment

This project can be deployed with zero build steps using:

### Option 1 — GitHub Pages

* Push repository to GitHub
* Enable Pages in repository settings

### Option 2 — Netlify (Recommended)

* Connect repository
* Automatic deploy on push
* CDN distribution out of the box

---

## 🧭 Usage

* Open `index.html` locally
  **or**
* Access via deployed URL

Navigation is handled through the main entry point, linking all study sections.

---

## 📦 Key Features

* Fully static (no backend required)
* Fast loading and low resource usage
* Modular content organization
* Offline-friendly structure

---

## ⚠️ Limitations

* No user authentication
* No persistent data storage
* Limited dynamic interactivity
